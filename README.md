# ✨ AI Agent 開發專案：PTT 八卦版熱門文章爬蟲

## 📝 專案介紹

這是一個由先進 AI Agent 自主設計與開發的實驗性專案，旨在探索其在資料收集、處理與分析領域的潛力。

**目前的任務**是針對台灣知名社群平台 PTT (Professional Technology Temple) 的 **八卦版 (Gossiping)** 進行熱門文章的資訊爬取。透過此任務，AI Agent 將學習如何與網站互動、解析 HTML 結構並提取目標資料。

## 🎯 目前任務與功能

此專案的 AI Agent 已完成了以下功能：

*   **目標網站：** PTT 八卦版 (Gossiping)。
*   **爬取內容：** 針對八卦版首頁的熱門文章，爬取以下資訊：
    *   **標題 (Title)：** 文章的完整標題。
    *   **推文數 (Push Count)：** 文章目前的推文數 (或標記為「爆」、「X1」等)。
    *   **作者 (Author)：** 發文者的 PTT ID。
*   **技術棧：**
    *   **Python 語言** 開發。
    *   使用 `requests` 函式庫發送 HTTP 請求，獲取網頁內容。
    *   使用 `BeautifulSoup4` (bs4) 進行 HTML 內容解析，提取所需資料。
*   **輸出形式：** 將爬取到的資料即時輸出顯示在終端機 (Console) 上。
